What is Data Liberation?

Liberated = Free Data

  • Unlocked
  • Widely supported (file) format
  • Transformable into other formats

Common Platforms

  • Allow export if legislation requires it
  • No common export file format
  • Exports often take hours or days, sometimes via support
  • Partial data (e.g. no friendship data)

Twitter / X: Not bad

WordPress as Liberation Target

  • Open Source

WordPress Importer Status Quo

Import from Platform Exports

  • Huge variety of file formats
  • Not well-defined (no standards)
  • Change over time
  • → Needs maintenance

Export File for WordPress

  • Platforms don't provide WordPress eXtended RSS files
  • Transform platform exports into WXR?
  • WXR format is text-only (no attachments)
  • Content-types not supported by WordPress natively → ?

WordPress Playground
to the rescue!

WordPress Playground

  • WordPress running in the browser
  • No server-side code running
  • Can run offline

Detour: Why is WordPress Playground a big deal?

(for you!)

WordPress Playground

  • Try plugins without risking your live site
  • Showcase a website before you have a host for it
  • Run beta versions of WordPress or plugins to test them
  • Translate a live plugin instead of guessing its UI

Playground as a Developer Tool

  • Query API: Set it up via URL
  • Javascript API: Embed and control it
  • Blueprints: Complex setups
Playground

Query API

  • playground.wordpress.net/?plugin=friends
  • playground.wordpress.net/?theme=tove
  • playground.wordpress.net/?language=nl_NL
  • playground.wordpress.net/?wp=nightly
  • playground.wordpress.net/?mode=seamless
  • ... ?multisite=yes ... ?url=/landingpage ... ?blueprint-url=
  • Combine like this: ?plugin=chatrix&theme=neve
Playground

Javascript API

  • Embed Playground and control it
  • Translate Live was created with it

Playground as a Demo Tool

  • Functional Preview of your Plugin
    • Add a blueprint to your plugin assets
  • Link a full demo WordPress
    • Configure with a blueprint:
      content, plugins, theme, ...
  • Embed WordPress in a block
    • WordPress Playground Block

Provide a Preview of Your Plugin

Playground

Blueprints

  • Pre-Configure WordPress with plugins, theme, site data
    • a single playground.wordpress.net link
  • Blueprint = JSON file with "steps", for example:
    { "step": "installPlugin", "slug": "friends" }
  • 27 different steps exist today
  • Visual Tool: Playground Step Library
    • provides even more steps

Examples: Blueprints Gallery

  • Feed Reader with the Friends Plugin
  • A Woo store with custom theme, content, and products
  • Use the Create Block Theme and start editing right away
  • WPGraphQL to query WordPress
  • Use wp-cli command to add posts
