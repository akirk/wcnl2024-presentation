What is Data Liberation?

Liberated = Free Data

  • Unlocked
  • Widely supported (file) format
  • Transformable into other formats

Common Platforms

  • Allow export if legislation requires it
  • No common export file format
  • Exports take time, sometimes through support
  • Partial data (e.g. no friendship data)

Why is WordPress Playground a solution for data liberation?

  • Can be easily reset → just try again!
  • Side-by-side view with a browser extension
  • Liberate data into a live WordPress
  • → Plugins can handle non-native types

How it works

  • Browser has access your data (logged in)
  • You tell the extension which data to liberate
  • Data gets directly sent to WordPress (Playground)

WordPress cannot handle natively

  • Contact Forms
  • Restaurant Menus
  • Products
  • ...

Plugins can handle that

  • Liberate Data = Free choice
  • Select your favorite plugin for XYZ
  • Fails? Try another plugin and try again

Building the extension

  • https://github.com/WordPress/try-wordpress
  • Currently: Firefox or Chrome
  • Loads playground.wordpress.net in the sidebar
  • Assisted export from any website

After the liberation

  • View and try it directly on playground.wp.net
  • Export to a hosted WordPress

How you can help

  • Share your assisted export
  • Review assisted exports
  • Make your plugin ready to receive liberated data
