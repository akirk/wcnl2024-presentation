What is Data Liberation?

It means the ability to use your data from one system in another system. Typically involves exporting and importing data.

Getting Data Out

Legislation may require companies to provide data portability. However, many companies do not prioritize this feature.

WordPress as a Transition Platform

WordPress has importers and exporters, making it a suitable platform for data migration.

WordPress Playground

A project that allows running WordPress in the browser using WASM, enabling safe plugin testing and more!

Use Cases for WordPress Playground

  • Try out plugins without risking your live site
  • Showcase a website before you have hosting
  • Run beta versions of WordPress or plugins

Experiment with Playground

Pre-configuring WordPress

Speed up the setup with query parameters for themes and plugins.

Building Blueprints

Create custom "startup definitions" to tailor your WordPress experience in Playground.

Where are we taking Playground?

Adapting Playground for CORS requests and translating MySQL to SQLite for more complex plugins.

How does Playground work?

Utilizes PHP compiled to run in WebAssembly with a local SQLite database stored in the browser.

WordPress Playground and Importing

Facilitates iteration in the import process, making it ideal for developers.
